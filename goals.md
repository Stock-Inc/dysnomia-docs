# Цели проекта

## Бизнес-цели

* Достичь минимальной базы активных пользователей для запуска премиум-функций

* Обеспечить 70%+ retention rate

* Подготовить инфраструктуру для бесперебойного общения

## Технические цели

* Быстрая доставка сообщений

* Высокий uptime сервиса

* End-to-end шифрование всех сообщений

* Горизонтальное масштабирование

* Использование современных технологий

## Пользовательские цели

* Интуитивный интерфейс

* Минимальная настройка

* Полный контроль над своими данными

* Все необходимые функции в одном приложении
