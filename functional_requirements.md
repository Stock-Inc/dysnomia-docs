## Обязательные функции

* Регистрация по почте с верификацией

* Сообщения с текстом, медиафайлами, статусами доставки

* Поиск пользователей

* Личные чаты и глобальный чат

* End-to-end шифрование, блокировка пользователей

* Push уведомления

## Дополнительные функции

* Голосовые звонки

* Стикеры и GIF

* Темы оформления

* Боты и API для разработчиков
